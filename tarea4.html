01 — Introducción

¿Qué es CSS3?

CSS (Cascading Style Sheets) es el lenguaje que controla la apariencia visual de las páginas web. La versión 3 introdujo módulos independientes como Flexbox, Grid, animaciones, variables y mucho más.

🎨
Lenguaje de estilos
CSS no es un lenguaje de programación. Define cómo se ven los elementos HTML: colores, tamaños, posiciones.
📁
Archivo .css
Los estilos se guardan en archivos .css y se vinculan al HTML con la etiqueta <link> dentro del <head>.
🌊
Cascada
Los estilos se aplican en cascada: si dos reglas aplican al mismo elemento, gana la más específica o la última declarada.
🔗
Separación de responsabilidades
HTML estructura el contenido, CSS lo estiliza. Mantenerlos separados hace el código más limpio y fácil de mantener.
📱
Responsive Design
Con CSS puedes adaptar el diseño a cualquier tamaño de pantalla usando media queries y unidades flexibles.
⚡
Tres formas de usar CSS
Externo (archivo .css), interno (<style> en el <head>) o en línea (atributo style="" en el elemento).
Tip: Siempre usa CSS externo en un archivo separado. Es la forma más ordenada, reutilizable y fácil de mantener.

02 — Fundamentos

Sintaxis y estructura de una regla CSS

Toda regla CSS tiene la misma estructura: un selector que apunta al elemento, y una o más declaraciones de propiedad y valor dentro de llaves.

Anatomía de una regla CSS:
selector { propiedad: valor; }

El selector indica a qué elementos HTML se aplica el estilo. La propiedad es la característica visual a modificar. El valor define cómo debe verse.
estilos.css
/* Regla CSS básica */
h1 {
  color: #c0392b;
  font-size: 2rem;
  font-weight: 700;
  text-align: center;
}

/* Múltiples propiedades */
p {
  font-size: 16px;
  line-height: 1.6;
  color: #444;
  margin-bottom: 1rem;
}

/* Varios selectores, mismo estilo */
h1, h2, h3 {
  font-family: 'Georgia', serif;
}
ComentariosSintaxis
Los comentarios en CSS van entre /* */. El navegador los ignora. Úsalos para documentar tu código.
/* Esto es un comentario */
UnidadesValores
px: píxeles fijos. rem: relativo al root. em: relativo al padre. %: porcentaje del contenedor. vw/vh: porcentaje del viewport.
EspecificidadCascada
El orden de prioridad es: !important > estilos en línea > IDs > clases > etiquetas. A mayor especificidad, mayor prioridad.
HerenciaCascada
Algunas propiedades como color y font-family se heredan de los elementos padre a los hijos automáticamente.

Los selectores son patrones que indican a qué elementos del HTML se aplicarán los estilos. CSS ofrece una gran variedad para apuntar a elementos con precisión.

04 — Modelo de caja

El modelo de caja (Box Model)

En CSS, todo elemento es una caja rectangular. El modelo de caja describe el espacio que ocupa: contenido, relleno, borde y margen.

Las 4 capas del Box Model (de adentro hacia afuera):
Content → el contenido real (texto, imagen) · Padding → espacio interno entre contenido y borde · Border → el borde del elemento · Margin → espacio externo entre este elemento y los demás
Vista previa · Box Model
margin
border
padding
content
marginEspaciado
Espacio exterior. Separa el elemento de sus vecinos. Acepta valores individuales o abreviados.
margin: 16px; /* todos lados */
margin: 8px 16px; /* arriba/abajo izq/der */
margin: 0 auto; /* centrar horizontalmente */
paddingEspaciado
Espacio interior entre el contenido y el borde. La misma sintaxis que margin.
padding: 20px; /* todos lados */
padding: 10px 20px; /* ver/hor */
padding-top: 8px; /* lado específico */
borderBorde
Define el borde del elemento. Se especifica grosor, estilo y color.
border: 1px solid #ccc;
border-radius: 8px;
border-top: 3px solid red;
box-sizingModelo
border-box hace que padding y border se incluyan en el width/height declarado. Recomendado siempre.
* {
box-sizing: border-box;
}
width / heightDimensiones
Definen el ancho y alto del área de contenido. Puedes usar px, %, rem, vw, vh o auto.
width: 300px;
max-width: 100%;
height: auto;
min-height: 200px;
overflowDesbordamiento
Controla qué pasa cuando el contenido supera el tamaño del contenedor: visible, hidden, scroll, auto.
overflow: hidden;
overflow-x: auto;
overflow-y: scroll;

05 — Tipografía

Propiedades tipográficas

CSS ofrece un control total sobre la tipografía: fuentes, tamaños, peso, espaciado e interlineado. Una buena tipografía es clave en el diseño web.

PropiedadValores comunesDescripción
font-family'Arial', sans-serifDefine la fuente. Siempre incluye una fuente genérica de respaldo.
font-size16px, 1rem, 1.2emTamaño del texto. Preferir rem para accesibilidad.
font-weight400 (normal), 700 (bold)Grosor del texto. Va de 100 a 900.
font-stylenormal, italic, obliqueEstilo de la fuente.
line-height1.5, 1.6, 24pxInterlineado. Sin unidad es multiplicador del font-size.
letter-spacing0.05em, 2pxEspaciado entre letras.
text-alignleft, center, right, justifyAlineación horizontal del texto.
text-decorationnone, underline, line-throughSubrayado y decoraciones del texto.
text-transformuppercase, lowercase, capitalizeTransformación de mayúsculas/minúsculas.
text-shadow2px 2px 4px rgba(0,0,0,.3)Sombra del texto: x y blur color.
tipografia.css
/* Importar Google Font */
@import url('https://fonts.googleapis.com/css2?family=Lora:wght@400;700&display=swap');

body {
  font-family: 'Lora', Georgia, serif;
  font-size: 16px;
  line-height: 1.7;
  color: #1c1713;
}

h1 {
  font-size: clamp(2rem, 5vw, 3.5rem);
  font-weight: 700;
  letter-spacing: -0.02em;
  text-transform: none;
}

06 — Colores y Fondos

Colores y propiedades de fondo

CSS admite múltiples formatos para definir colores y ofrece poderosas propiedades para estilizar los fondos de los elementos.

colorTexto
Define el color del texto. Admite nombres, hexadecimal, RGB, RGBA, HSL.
color: red;
color: #c0392b;
color: rgb(192, 57, 43);
color: rgba(0,0,0, 0.5);
color: hsl(5, 63%, 46%);
backgroundFondo
Propiedad abreviada para definir el fondo: color, imagen, posición, tamaño y repetición.
background: #faf9f7;
background: url('img.jpg') center/cover no-repeat;
background: linear-gradient(135deg, #red, #blue);
opacityTransparencia
Transparencia del elemento completo (incluye hijos). Va de 0 (invisible) a 1 (opaco). Distinto de rgba.
opacity: 1; /* opaco */
opacity: 0.5; /* 50% transparente */
opacity: 0; /* invisible */
box-shadowSombra
Sombra de la caja: desplazamiento x/y, desenfoque, extensión y color. Se pueden combinar múltiples sombras.
box-shadow: 0 4px 16px rgba(0,0,0,.1);
box-shadow: inset 0 2px 4px rgba(0,0,0,.06);
gradientDegradado
Fondos degradados sin imagen. Linear (lineal) y radial (circular). Se aplica como valor de background.
background: linear-gradient(to right, #c0392b, #e74c3c);
background: radial-gradient(circle, #fff, #eee);
filterFiltros
Efectos visuales sobre el elemento: blur, brightness, contrast, grayscale, saturate, etc.
filter: blur(4px);
filter: brightness(0.85);
filter: grayscale(100%);

07 — Display y Posición

Display y posicionamiento

La propiedad display define cómo se comporta un elemento en el flujo del documento. position controla su ubicación en la página.

display: blockDisplay
Ocupa todo el ancho disponible y fuerza un salto de línea. Comportamiento por defecto de <div>, <p>, <h1>, etc.
display: inlineDisplay
Solo ocupa el espacio de su contenido, no genera salto de línea. No acepta width/height. Por defecto en <span>, <a>, <strong>.
display: inline-blockDisplay
Combinación: en línea con otros elementos pero acepta width, height, padding y margin verticales.
display: noneDisplay
Oculta el elemento completamente, como si no existiera. No ocupa espacio. Distinto de visibility: hidden.
position: relativePosición
El elemento se posiciona relativo a su lugar original en el flujo. Usa top, right, bottom, left para desplazarlo.
position: relative;
top: 10px;
left: 20px;
position: absolutePosición
Se posiciona respecto al ancestro más cercano con position distinto de static. Sale del flujo normal.
position: absolute;
top: 0;
right: 0;
position: fixedPosición
Se posiciona respecto al viewport (pantalla). Permanece fijo al hacer scroll. Ideal para navbars y botones flotantes.
position: fixed;
top: 0;
width: 100%;
z-indexCapas
Controla el orden de apilamiento de elementos posicionados. Mayor número = encima. Solo funciona con position distinto de static.
z-index: 10;
z-index: 100;
z-index: -1; /* detrás */

08 — Flexbox

CSS Flexbox

Flexbox es un modelo de diseño que permite distribuir y alinear elementos en una dimensión (fila o columna) de forma flexible y sencilla.

Cómo activarlo: Aplicar display: flex al contenedor padre convierte a todos sus hijos directos en "flex items" y activa el contexto flex.
flexbox.css
/* Contenedor flex */
.contenedor {
  display: flex;
  flex-direction: row;         /* row | column | row-reverse */
  justify-content: space-between; /* eje principal */
  align-items: center;          /* eje secundario */
  flex-wrap: wrap;              /* permitir salto de línea */
  gap: 16px;                    /* espacio entre items */
}

/* Item flex */
.item {
  flex: 1;       /* crece para llenar espacio */
  flex: 0 0 200px; /* no crece, no encoge, base 200px */
  align-self: flex-start; /* alineación individual */
}

flex-directionrow, column, row-reverse, column-reverseContenedor
justify-contentflex-start, flex-end, center, space-between, space-aroundContenedor
align-itemsstretch, flex-start, flex-end, center, baselineContenedor
flex-wrapnowrap, wrap, wrap-reverseContenedor
gap16px, 1rem, 8px 16pxContenedor

09 — CSS Grid

CSS Grid

Grid es el sistema de maquetación bidimensional de CSS. Permite crear layouts complejos con filas y columnas de forma declarativa y precisa.

grid.css
/* Contenedor grid */
.grid {
  display: grid;
  grid-template-columns: repeat(3, 1fr);   /* 3 columnas iguales */
  grid-template-rows: auto 1fr auto;
  gap: 24px;
}

/* Columnas responsive sin media queries */
.grid-auto {
  display: grid;
  grid-template-columns: repeat(auto-fill, minmax(280px, 1fr));
  gap: 1.5rem;
}

/* Item que ocupa varias columnas */
.item-ancho {
  grid-column: 1 / 3;    /* de columna 1 a 3 */
  grid-row: span 2;      /* ocupa 2 filas */
}
fr (fracción)Unidad
Unidad exclusiva de Grid. Representa una fracción del espacio disponible. 1fr 2fr = una parte y dos partes.
repeat()Función
Atajo para repetir columnas o filas. repeat(4, 1fr) = cuatro columnas iguales.
minmax()Función
Define un rango de tamaño mínimo y máximo. minmax(200px, 1fr) = mínimo 200px, máximo lo que alcance.
auto-fill / auto-fitPalabras clave
Con minmax crean layouts responsivos automáticamente. auto-fill mantiene columnas vacías; auto-fit las colapsa.

10 — Transiciones

Transiciones CSS

Las transiciones permiten animar suavemente el cambio de un valor de propiedad a otro, como al hacer hover. Se configuran en el estado inicial del elemento.

transiciones.css
.boton {
  background: #c0392b;
  color: white;
  padding: 12px 28px;
  border-radius: 6px;

  /* Transición: propiedad duración timing */
  transition: background 0.25s ease, transform 0.2s ease;
}

.boton:hover {
  background: #922b21;
  transform: translateY(-2px);
}

/* Transición de todas las propiedades */
.card {
  transition: all 0.3s ease;
}
PropiedadDescripción
transition-propertyQué propiedad animar (all, color, transform...)
transition-durationDuración de la animación (0.3s, 300ms)
transition-timing-functionCurva de velocidad: ease, linear, ease-in, ease-out, ease-in-out, cubic-bezier()
transition-delayEsperar antes de iniciar la transición (0.1s, 100ms)
transition (abreviado)propiedad duración timing delay

11 — Animaciones

Animaciones con @keyframes

Las animaciones CSS permiten crear movimientos complejos con múltiples pasos, sin necesidad de JavaScript, usando @keyframes y la propiedad animation.

animaciones.css
/* 1. Definir la animación con @keyframes */
@keyframes aparecer {
  from {
    opacity: 0;
    transform: translateY(20px);
  }
  to {
    opacity: 1;
    transform: translateY(0);
  }
}

/* 2. Aplicarla al elemento */
.tarjeta {
  animation: aparecer 0.6s ease both;
}

/* Múltiples pasos con porcentajes */
@keyframes pulsar {
  0%   { transform: scale(1); }
  50%  { transform: scale(1.05); }
  100% { transform: scale(1); }
}

.icono {
  animation: pulsar 2s ease-in-out infinite;
}
PropiedadDescripción
animation-nameNombre del @keyframes a usar
animation-durationCuánto dura un ciclo (1s, 500ms)
animation-timing-functionCurva de velocidad: ease, linear, steps()
animation-delayEspera antes de iniciar
animation-iteration-countVeces que se repite: 1, 3, infinite
animation-directionnormal, reverse, alternate, alternate-reverse
animation-fill-modeEstado antes/después: none, forwards, backwards, both
animation (abreviado)nombre duración timing delay iteraciones dirección fill-mode

12 — Responsive Design

Diseño Responsivo y Media Queries

El responsive design adapta la apariencia de una página a distintos tamaños de pantalla. CSS lo logra principalmente con media queries y unidades relativas.

responsive.css
/* Estilos base (mobile first) */
.grid {
  display: grid;
  grid-template-columns: 1fr;  /* 1 columna en móvil */
  gap: 1rem;
}

/* Tablet: 768px en adelante */
@media (min-width: 768px) {
  .grid {
    grid-template-columns: repeat(2, 1fr);
  }
}

/* Escritorio: 1024px en adelante */
@media (min-width: 1024px) {
  .grid {
    grid-template-columns: repeat(3, 1fr);
  }
  .sidebar {
    display: block;
  }
}

/* Preferencia de movimiento reducido */
@media (prefers-reduced-motion: reduce) {
  * { animation: none; transition: none; }
}
Mobile First: Diseña primero para pantallas pequeñas y luego agrega estilos para pantallas más grandes con min-width. Es la estrategia recomendada actualmente.

13 — Variables CSS

Variables CSS (Custom Properties)

Las variables CSS permiten definir valores reutilizables en un solo lugar. Son ideales para paletas de colores, tipografías y espaciados consistentes en todo el proyecto.

variables.css
/* Definir variables en :root (global) */
:root {
  --color-primario: #c0392b;
  --color-texto: #1c1713;
  --fuente-display: 'Lora', Georgia, serif;
  --espacio-md: 1rem;
  --radio-borde: 8px;
  --sombra: 0 4px 16px rgba(0,0,0,.07);
}

/* Usar variables con var() */
.boton {
  background: var(--color-primario);
  font-family: var(--fuente-display);
  padding: var(--espacio-md);
  border-radius: var(--radio-borde);
  box-shadow: var(--sombra);
}

/* Sobreescribir en un contexto específico */
.tema-oscuro {
  --color-texto: #ede0e0;
  --color-primario: #ff6b6b;
}
✅
Consistencia
Cambias una variable y se actualiza en todo el sitio automáticamente.
✅
Legibilidad
var(--color-primario) es más descriptivo que #c0392b.
✅
Temas de color
Cambiar entre modo claro y oscuro es tan simple como cambiar los valores de las variables.
✅
Valor de respaldo
var(--color, red) usa "red" si la variable no está definida.

14 — Práctica

Ejemplo completo

Una hoja de estilos completa que aplica todo lo aprendido: variables, tipografía, modelo de caja, flexbox, grid, transiciones y responsive design.

portafolio.css
/* ── Variables globales ── */
:root {
  --bg: #0a0808;
  --surface: #160f0f;
  --accent: #ff3333;
  --text: #ede0e0;
  --muted: #8a6060;
  --border: #2a1515;
  --font: 'Syne', sans-serif;
  --shadow: 0 4px 20px rgba(0,0,0,.4);
}

/* ── Reset base ── */
*, *::before, *::after {
  box-sizing: border-box;
  margin: 0;
  padding: 0;
}

body {
  background: var(--bg);
  color: var(--text);
  font-family: var(--font);
  line-height: 1.7;
}

/* ── Layout con Grid ── */
.cards-grid {
  display: grid;
  grid-template-columns: repeat(auto-fill, minmax(300px, 1fr));
  gap: 1.5rem;
  padding: 2rem;
}

/* ── Card con Flexbox ── */
.card {
  background: var(--surface);
  border: 1px solid var(--border);
  border-radius: 12px;
  display: flex;
  flex-direction: column;
  overflow: hidden;
  box-shadow: var(--shadow);
  transition: transform 0.3s ease, border-color 0.3s ease;
}

.card:hover {
  transform: translateY(-6px);
  border-color: var(--accent);
}

/* ── Responsive ── */
@media (max-width: 640px) {
  .cards-grid {
    grid-template-columns: 1fr;
    padding: 1rem;
  }
}
