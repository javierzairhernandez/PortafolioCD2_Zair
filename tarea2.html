Los lenguajes web son la base de todo lo que existe en internet. Se dividen en tecnologías de front-end (interfaz de usuario) y back-end (lógica del servidor), aunque varios operan en ambos contextos.

HTML5

Lenguaje de marcado estándar para estructurar el contenido de las páginas web. Define la semántica y jerarquía de la información.

MarcadoFront-endSemántico

  <body>

    <p>Bienvenido a la web.</p>
  </body>
</html>

Los lenguajes de escritorio permiten crear aplicaciones que corren directamente en el sistema operativo, con acceso al hardware y mayor potencia que las soluciones web. Son la base del software profesional y de sistemas.

Python

Lenguaje de propósito general con sintaxis limpia. Líder en IA, ciencia de datos y automatización. Sus interfaces de escritorio se construyen con Tkinter o PyQt.

IA / MLScriptingDatos
app.py
# Clase y herencia en Python
class Animal:
    def __init__(self, nombre):
        self.nombre = nombre

    def hablar(self):
        return f"Soy {self.nombre}"

perro = Animal("Rex")
print(perro.hablar())

Los lenguajes y herramientas de diseño no siempre son lenguajes de programación convencionales, pero sí son sistemas formales con sintaxis propia que determinan la apariencia, estructura y presentación del contenido visual y editorial.

SASS / SCSS

Preprocesador CSS que añade variables, anidamiento, mixins y funciones. Compilado a CSS puro, permite hojas de estilo más mantenibles y escalables.

PreprocesadorVariablesMixins
_components.scss
// Mixin reutilizable con parámetro
@mixin flex-center($dir: row) {
  display: flex;
  justify-content: center;
  align-items: center;
  flex-direction: $dir;
}
.hero {
  @include flex-center(column);
}
