01 // INTRODUCCIÓN

🐍 ¿Qué es Python?

Python es un lenguaje de programación de alto nivel, interpretado y de propósito general creado por Guido van Rossum y publicado en 1991. Su filosofía de diseño enfatiza la legibilidad del código y permite a los programadores expresar conceptos en menos líneas que en otros lenguajes.

📖
Legible
Sintaxis clara que se parece al inglés natural. Fácil de aprender y mantener.
⚡
Versátil
Web, IA, ciencia de datos, automatización, scripting y mucho más.
🌍
Comunidad
Una de las comunidades más grandes del mundo. Miles de librerías disponibles.
🔓
Open Source
Completamente libre y gratuito para cualquier uso comercial o personal.

02 // VARIABLES Y TIPOS DE DATOS

📦 Variables y Tipos de Datos

En Python, las variables son contenedores que almacenan valores. No es necesario declarar el tipo, Python lo infiere automáticamente (tipado dinámico).

Tipos de datos primitivos en Python
variables.py Python 
# Tipos básicos de datos
nombre   = "Ana García"      # str  — cadena de texto
edad     = 25               # int  — número entero
altura   = 1.68             # float — número decimal
activo   = True             # bool — verdadero/falso
dato     = None             # NoneType — sin valor

# Verificar el tipo con type()
print(type(nombre))   # <class 'str'>
print(type(edad))     # <class 'int'>

# Múltiple asignación en una línea
x, y, z = 10, 20, 30

# Conversión de tipos (casting)
numero_texto = "42"
numero_int   = int(numero_texto)   # "42" → 42
numero_float = float(99)           # 99 → 99.0
texto_num    = str(3.14)           # 3.14 → "3.14"

03 // ESTRUCTURAS DE CONTROL

🔀 Estructuras de Control

Las estructuras de control dirigen el flujo de ejecución del programa según condiciones y repeticiones.

Condicionales — if / elif / else

Python usa la indentación (espacios) para delimitar bloques, no llaves. Esto obliga a escribir código limpio.

condicionales.py Python 
edad = 20

if edad >= 18:
    print("Eres mayor de edad")
elif edad >= 13:
    print("Eres adolescente")
else:
    print("Eres un niño")

# Operadores de comparación
# ==  igual a        !=  diferente de
# >   mayor que       <   menor que
# >=  mayor o igual   <=  menor o igual

# Operadores lógicos
x = 15
if x > 10 and x < 20:
    print("x está entre 10 y 20")

# Condicional en una línea (ternario)
estado = "adulto" if edad >= 18 else "menor"
▶ salida
Eres mayor de edad
x está entre 10 y 20

Bucle for

Itera sobre cualquier secuencia: listas, rangos, cadenas, diccionarios, etc.

bucle_for.py Python 
# Iterar sobre un rango de números
for i in range(1, 6):
    print(f"Número: {i}")

# Iterar sobre una lista
frutas = ["manzana", "naranja", "mango"]
for fruta in frutas:
    print(f"Fruta: {fruta}")

# enumerate — índice + valor
for i, fruta in enumerate(frutas):
    print(f"{i}. {fruta}")

# Comprensión de lista (list comprehension)
cuadrados = [x**2 for x in range(1, 6)]
print(cuadrados)  # [1, 4, 9, 16, 25]
▶ salida
Número: 1 ... Número: 5
Fruta: manzana — naranja — mango
0. manzana 1. naranja 2. mango
[1, 4, 9, 16, 25]

Bucle while

bucle_while.py Python 
contador = 0

while contador < 5:
    print(f"Contador: {contador}")
    contador += 1   # equivale a contador = contador + 1

# break — salir del bucle
# continue — saltar a la siguiente iteración
for n in range(10):
    if n == 3: continue   # salta el 3
    if n == 7: break     # para en el 7
    print(n, end=" ")    # 0 1 2 4 5 6
⚠️ Importante: Python usa indentación (4 espacios o 1 tab) para definir bloques de código. Un error de indentación es un error de sintaxis.
04 // FUNCIONES

⚙️ Funciones

Las funciones son bloques de código reutilizables que realizan una tarea específica. Se definen con la palabra clave def.

Flujo de una función: entrada → procesamiento → salida
funciones.py Python 
# Función básica
def saludar(nombre):
    return f"¡Hola, {nombre}!"

print(saludar("María"))   # ¡Hola, María!

# Parámetros con valor por defecto
def potencia(base, exponente=2):
    return base ** exponente

print(potencia(4))       # 16  (usa exp=2 por defecto)
print(potencia(2, 10))   # 1024

# *args — número variable de argumentos
def sumar(*numeros):
    return sum(numeros)

print(sumar(1, 2, 3, 4, 5))  # 15

# **kwargs — argumentos clave-valor
def perfil(**datos):
    for clave, valor in datos.items():
        print(f"{clave}: {valor}")

perfil(nombre="Ana", edad=25, ciudad="CDMX")

# Lambda — función anónima de una línea
doble = lambda x: x * 2
print(doble(7))   # 14
▶ salida
¡Hola, María!
16
1024
15
nombre: Ana | edad: 25 | ciudad: CDMX
14

# ── LIST ─────────────────────────────
frutas = ["manzana", "banana", "cereza"]
frutas.append("mango")      # agrega al final

print(frutas[0])            # manzana (índice 0)
print(frutas[-1])           # mango   (último)

# ── TUPLE ────────────────────────────
coordenadas = (19.43, -99.13)  # no se puede modificar

    "ciudad": "Guadalajara"

print(persona["nombre"])         # Luis
persona["email"] = "l@mail.com" # agregar clave
print(persona.get("telefono", "N/A"))  # N/A si no existe

# ── SET ──────────────────────────────
colores = {"rojo", "azul", "verde", "rojo"}
print(colores)  # {'rojo', 'azul', 'verde'} — sin duplicados

06 // CADENAS DE TEXTO

💬 Cadenas de Texto (Strings)

Las cadenas son uno de los tipos más usados en Python. Ofrecen una riqueza de métodos para transformar, buscar y formatear texto.

strings.py Python 
texto = "  Hola, Mundo Python  "

# Métodos de transformación
print(texto.upper())       # HOLA, MUNDO PYTHON
print(texto.lower())       # hola, mundo python
print(texto.strip())       # "Hola, Mundo Python"
print(texto.replace("Mundo", "Universo"))

# Búsqueda
print("Python" in texto)   # True
print(texto.find("Mundo")) # índice donde empieza
print(texto.count("o"))    # cuántas veces aparece

# Split y join
palabras = "uno,dos,tres".split(",")
print(palabras)              # ['uno', 'dos', 'tres']
print(" - ".join(palabras))  # uno - dos - tres

# f-strings (formato moderno, recomendado)
nombre = "Carlos"
edad   = 28
pi     = 3.14159
print(f"{nombre} tiene {edad} años")
print(f"Pi ≈ {pi:.2f}")        # 2 decimales → 3.14
print(f"{nombre.upper()}")    # expresiones dentro

# Indexación y slicing
s = "Python"
print(s[0])      # P
print(s[-1])     # n
print(s[0:3])    # Pyt
print(s[::-1])   # nohtyP  (invertida)
▶ salida
HOLA, MUNDO PYTHON
Hola, Universo Python
True
['uno', 'dos', 'tres'] → uno - dos - tres
Carlos tiene 28 años
Pi ≈ 3.14
CARLOS
07 // PROGRAMACIÓN ORIENTADA A OBJETOS

🏗️ Clases y Objetos (POO)

La Programación Orientada a Objetos organiza el código en clases que modelan entidades del mundo real, con sus atributos (datos) y métodos (comportamientos).

Clase como plantilla → múltiples objetos (instancias)
clases.py Python 
class Perro:
    """Clase que representa a un perro."""

    # Variable de clase (compartida por todas las instancias)
    especie = "Canis familiaris"

    def __init__(self, nombre, raza, edad):
        # Atributos de instancia
        self.nombre = nombre
        self.raza   = raza
        self.edad   = edad

    def ladrar(self):
        return f"{self.nombre} dice: ¡Guau!"

    def descripcion(self):
        return f"{self.nombre} es un {self.raza} de {self.edad} años"

    def __str__(self):   # representación en texto
        return f"Perro({self.nombre})"


# Crear instancias
rex  = Perro("Rex",  "Labrador", 3)
luna = Perro("Luna", "Poodle",   5)

print(rex.ladrar())
print(luna.descripcion())
print(Perro.especie)


# ── HERENCIA ─────────────────────────
class PerroGuia(Perro):       # hereda de Perro
    def __init__(self, nombre, dueno):
        super().__init__(nombre, "Golden", 2)
        self.dueno = dueno

    def guiar(self):             # método nuevo
        return f"{self.nombre} guía a {self.dueno}"

guia = PerroGuia("Buddy", "Pedro")
print(guia.guiar())
print(guia.ladrar())  # hereda el método
▶ salida
Rex dice: ¡Guau!
Luna es un Poodle de 5 años
Canis familiaris
Buddy guía a Pedro
Buddy dice: ¡Guau!

09 // MANEJO DE ERRORES

🛡️ Manejo de Errores y Excepciones

Los errores en Python se llaman excepciones. Con try / except puedes capturarlas y manejarlas sin que el programa se detenga bruscamente.

ExcepciónCuándo ocurre
TypeErrorOperación con tipos incompatibles (ej: "2" + 2)
ValueErrorValor incorrecto para una función (ej: int("abc"))
IndexErrorÍndice fuera de rango en una lista
KeyErrorClave inexistente en un diccionario
ZeroDivisionErrorDivisión entre cero
FileNotFoundErrorArchivo no encontrado
AttributeErrorAtributo o método inexistente en un objeto
excepciones.py Python 
# Estructura básica try / except
try:
    resultado = 10 / 0
except ZeroDivisionError:
    print("Error: no se puede dividir entre cero")

# Múltiples excepciones
def convertir(valor):
    try:
        return int(valor)
    except ValueError:
        print(f"'{valor}' no es un número válido")
        return None
    except TypeError:
        print("Tipo de dato incorrecto")
        return None

print(convertir("42"))    # 42
print(convertir("abc"))   # Error + None

# try / except / else / finally
try:
    archivo = open("datos.txt", "r")
    contenido = archivo.read()
except FileNotFoundError:
    print("Archivo no encontrado")
else:
    print("Archivo leído exitosamente")   # si no hubo error
finally:
    print("Esto siempre se ejecuta")      # siempre

# Lanzar excepciones personalizadas
def validar_edad(edad):
    if edad < 0:
        raise ValueError(f"Edad inválida: {edad}")
    return edad

try:
    validar_edad(-5)
except ValueError as e:
    print(f"Capturado: {e}")
▶ salida
Error: no se puede dividir entre cero
42
'abc' no es un número válido
Archivo no encontrado
Esto siempre se ejecuta
Capturado: Edad inválida: -5
✅ Buena práctica: Siempre captura excepciones específicas en lugar de un except genérico. Esto hace tu código más predecible y fácil de depurar.
